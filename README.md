## 🚀 How to Deploy
1. Clone this repository or download the ZIP.  
2. Push to a GitHub repository named `agustinperalta.github.io`.  
3. Enable GitHub Pages under **Settings > Pages > Source**.  
4. Your website will be live in a few minutes.
